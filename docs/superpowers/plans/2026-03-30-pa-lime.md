# PA-LIME (Protocol-Aware LIME) Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a standalone, reusable Python library (`pa_lime`) that provides Protocol-Aware Local Interpretable Model-Agnostic Explanations for auditing NIDS models, enforcing physical network protocol invariants during perturbation sampling.

**Architecture:** The library is structured into four layers: (1) dataset schema definitions that encode feature names, types, and protocol invariants for CIC-IDS-2017, CSE-CIC-IDS2018, NSL-KDD, UNSW-NB15 (both native and CICFlowMeter), plus a user-extensible custom schema API; (2) a Domain-Constraint Tabular Fuzzer that generates vectorized neighborhoods honoring non-negativity, hierarchical ordering, discrete integrity, and protocol-gating constraints; (3) a LIME explainer that fits weighted Ridge surrogates and returns rich result objects with coefficients, fidelity, intercept, and feature names; (4) evaluation metrics including semantic robustness. Auto-detection of protocol encoding (string vs integer) is handled at the schema level.

**Tech Stack:** Python 3.12+, NumPy, scikit-learn (Ridge, StandardScaler, pairwise_distances), SciPy (spearmanr), dataclasses

# pa_lime/result.py
"""Rich result objects for PA-LIME explanations."""

from dataclasses import dataclass, field

import numpy as np


@dataclass(frozen=True)
class ExplanationResult:
    """Result of a single PA-LIME explanation.

    Attributes:
        feature_names: Ordered list of feature names matching coefficient indices.
        coefficients: Ridge surrogate coefficients (feature importances). Shape: (d,).
        intercept: Ridge surrogate intercept term.
        r_squared: R-squared fidelity of the surrogate on the weighted neighborhood.
        predicted_class: The class index or label being explained.
        local_prediction: The surrogate's prediction at the query point.
        num_samples: Number of neighborhood samples used.
    """

    feature_names: list[str]
    coefficients: np.ndarray
    intercept: float
    r_squared: float
    predicted_class: int | str | None
    local_prediction: float
    num_samples: int

    def top_features(self, k: int = 10, absolute: bool = True) -> list[tuple[str, float]]:
        """Return the top-k most important features by coefficient magnitude.

        Args:
            k: Number of features to return.
            absolute: If True, rank by absolute value. If False, rank by raw value.

        Returns:
            List of (feature_name, coefficient) tuples sorted descending.
        """
        if absolute:
            order = np.argsort(np.abs(self.coefficients))[::-1]
        else:
            order = np.argsort(self.coefficients)[::-1]
        return [(self.feature_names[i], float(self.coefficients[i])) for i in order[:k]]

    def as_dict(self) -> dict[str, float]:
        """Return feature_name -> coefficient mapping."""
        return dict(zip(self.feature_names, self.coefficients.tolist()))

# pa_lime/schemas.py
"""Dataset schema definitions and protocol-aware feature metadata for NIDS datasets."""

from __future__ import annotations

from dataclasses import dataclass, field

import numpy as np

# Integer protocol numbers per IANA
TCP_PROTOCOL_INT = 6
UDP_PROTOCOL_INT = 17
ICMP_PROTOCOL_INT = 1

# Label-encoded values used in NSL-KDD after standard preprocessing
# tcp -> some int, udp -> some int, icmp -> some int
# The actual encoding depends on the user's preprocessing; we detect at runtime.
_STRING_ENCODED_PROTOCOL_SENTINEL = {0, 1, 2}


@dataclass(frozen=True)
class HierarchicalConstraint:
    """Defines a Max >= Mean >= Min ordering constraint across three features.

    Attributes:
        max_feature: Feature name for the maximum value.
        mean_feature: Feature name for the mean value.
        min_feature: Feature name for the minimum value.
    """

    max_feature: str
    mean_feature: str
    min_feature: str


@dataclass
class DatasetSchema:
    """Schema describing the feature space and protocol invariants of a NIDS dataset.

    Attributes:
        name: Human-readable dataset name.
        feature_names: Ordered list of feature column names (excluding label).
        protocol_feature: Name of the column encoding the network protocol, or None.
        non_negative_features: Features that must be >= 0 (durations, counts, lengths).
        tcp_only_features: Features that must be 0 for non-TCP flows.
        discrete_features: Features that must remain integers (flag counts, packet counts).
        hierarchical_constraints: List of Max >= Mean >= Min ordering rules.
        protocol_encoding: How the protocol column is encoded.
            'integer' — IANA numbers (6=TCP, 17=UDP, 1=ICMP).
            'string' — label-encoded categorical (e.g., NSL-KDD's tcp/udp/icmp mapped to ints).
            'auto' — detect at explain-time from the instance values.
    """

    name: str
    feature_names: list[str]
    protocol_feature: str | None
    non_negative_features: list[str]
    tcp_only_features: list[str]
    discrete_features: list[str]
    hierarchical_constraints: list[HierarchicalConstraint]
    protocol_encoding: str  # "integer", "string", or "auto"

    # Cached index lookups (computed in __post_init__)
    protocol_index: int | None = field(init=False, default=None)
    non_negative_indices: list[int] = field(init=False, default_factory=list)
    tcp_only_indices: list[int] = field(init=False, default_factory=list)
    discrete_indices: list[int] = field(init=False, default_factory=list)
    hierarchical_index_triples: list[tuple[int, int, int]] = field(
        init=False, default_factory=list
    )

    def __post_init__(self) -> None:
        valid_encodings = {"integer", "string", "auto"}
        if self.protocol_encoding not in valid_encodings:
            raise ValueError(
                f"protocol_encoding must be one of {valid_encodings}, "
                f"got '{self.protocol_encoding}'"
            )

        name_to_idx = {name: i for i, name in enumerate(self.feature_names)}

        if self.protocol_feature is not None and self.protocol_feature in name_to_idx:
            self.protocol_index = name_to_idx[self.protocol_feature]

        self.non_negative_indices = [
            name_to_idx[n] for n in self.non_negative_features if n in name_to_idx
        ]
        self.tcp_only_indices = [
            name_to_idx[n] for n in self.tcp_only_features if n in name_to_idx
        ]
        self.discrete_indices = [
            name_to_idx[n] for n in self.discrete_features if n in name_to_idx
        ]

        triples = []
        for hc in self.hierarchical_constraints:
            max_i = name_to_idx.get(hc.max_feature)
            mean_i = name_to_idx.get(hc.mean_feature)
            min_i = name_to_idx.get(hc.min_feature)
            if max_i is not None and mean_i is not None and min_i is not None:
                triples.append((max_i, mean_i, min_i))
        self.hierarchical_index_triples = triples


def detect_protocol_encoding(
    row: np.ndarray,
    protocol_feature: str,
    feature_names: list[str],
) -> str:
    """Auto-detect whether the protocol column uses IANA integer or label-encoded string values.

    Heuristic:
      - If value is in {1, 6, 17} -> 'integer' (IANA protocol numbers for ICMP, TCP, UDP)
      - If value is in {0, 1, 2} -> 'string' (typical label encoding of tcp/udp/icmp)
      - Otherwise -> 'integer' (default: treat as IANA)

    Args:
        row: A single data instance as a 1D numpy array.
        protocol_feature: Name of the protocol column.
        feature_names: Ordered feature names matching row indices.

    Returns:
        'integer' or 'string'
    """
    idx = feature_names.index(protocol_feature)
    val = int(round(row[idx]))
    iana_values = {TCP_PROTOCOL_INT, UDP_PROTOCOL_INT, ICMP_PROTOCOL_INT}
    if val in iana_values:
        return "integer"
    if val in _STRING_ENCODED_PROTOCOL_SENTINEL:
        return "string"
    return "integer"


def _is_tcp(protocol_value: float, encoding: str, tcp_int: int = TCP_PROTOCOL_INT) -> bool:
    """Check if a protocol value represents TCP.

    Args:
        protocol_value: The numeric value in the protocol column.
        encoding: 'integer' or 'string'.
        tcp_int: The integer value representing TCP. For 'integer' encoding, this is 6 (IANA).
            For 'string' encoding, the user must provide the label-encoded TCP value.

    Returns:
        True if the protocol is TCP.
    """
    return int(round(protocol_value)) == tcp_int

def get_schema(name: str) -> DatasetSchema:
    """Look up a built-in schema by name.

    Args:
        name: One of 'CIC-IDS-2017', 'CSE-CIC-IDS2018', 'NSL-KDD',
              'UNSW-NB15-Native', 'UNSW-NB15-CICFlowMeter'.

    Returns:
        The corresponding DatasetSchema.

    Raises:
        KeyError: If the name is not a built-in schema.
    """
    if name not in BUILTIN_SCHEMAS:
        raise KeyError(
            f"Unknown schema '{name}'. Available: {list(BUILTIN_SCHEMAS.keys())}"
        )
    return BUILTIN_SCHEMAS[name]

# pa_lime/constraints.py
"""Vectorized constraint enforcement for protocol-aware neighborhood generation."""

from __future__ import annotations

import numpy as np

from pa_lime.schemas import DatasetSchema, TCP_PROTOCOL_INT


class ConstraintEnforcer:
    """Applies physical network protocol invariants to a neighborhood matrix.

    All operations are vectorized (NumPy broadcast/mask) for O(1) complexity
    relative to the number of constraints (linear in num_samples).

    Args:
        schema: The DatasetSchema defining which constraints apply.
    """

    def __init__(self, schema: DatasetSchema) -> None:
        self.schema = schema

    def enforce(
        self,
        neighborhood: np.ndarray,
        protocol_value: float | None,
        protocol_encoding: str,
        tcp_label_value: int | float = TCP_PROTOCOL_INT,
    ) -> np.ndarray:
        """Apply all constraints in-place and return the neighborhood.

        Constraint order:
          1. Fix protocol column to original value
          2. Non-negativity clamping
          3. Hierarchical ordering (min <= mean <= max)
          4. Discrete rounding
          5. Protocol-gated TCP feature zeroing

        Args:
            neighborhood: Shape (N, D) matrix of perturbed samples.
            protocol_value: The original instance's protocol value, or None if no protocol column.
            protocol_encoding: 'integer' or 'string'.
            tcp_label_value: The numeric value representing TCP in the protocol column.
                For 'integer' encoding, this is 6 (IANA). For 'string' (label-encoded),
                the caller must provide the integer that maps to TCP.

        Returns:
            The constrained neighborhood (modified in-place).
        """
        s = self.schema

        # 1. Fix protocol column
        if s.protocol_index is not None and protocol_value is not None:
            neighborhood[:, s.protocol_index] = protocol_value

        # 2. Non-negativity
        if s.non_negative_indices:
            idx = s.non_negative_indices
            neighborhood[:, idx] = np.maximum(0, neighborhood[:, idx])

        # 3. Hierarchical: mean = max(mean, min); max = max(max, mean)
        for max_i, mean_i, min_i in s.hierarchical_index_triples:
            neighborhood[:, mean_i] = np.maximum(
                neighborhood[:, mean_i], neighborhood[:, min_i]
            )
            neighborhood[:, max_i] = np.maximum(
                neighborhood[:, max_i], neighborhood[:, mean_i]
            )

        # 4. Discrete rounding
        if s.discrete_indices:
            idx = s.discrete_indices
            neighborhood[:, idx] = np.round(neighborhood[:, idx])

        # 5. Protocol-gated TCP feature zeroing
        if (
            s.tcp_only_indices
            and protocol_value is not None
        ):
            is_tcp = int(round(protocol_value)) == int(tcp_label_value)
            if not is_tcp:
                neighborhood[:, s.tcp_only_indices] = 0.0

        return neighborhood
```

# pa_lime/fuzzer.py
"""Domain-Constraint Tabular Fuzzer for protocol-aware neighborhood generation."""

from __future__ import annotations

import numpy as np

from pa_lime.constraints import ConstraintEnforcer
from pa_lime.schemas import (
    DatasetSchema,
    TCP_PROTOCOL_INT,
    detect_protocol_encoding,
)


class DomainConstraintFuzzer:
    """Generates constrained neighborhoods for NIDS flow data.

    Replaces standard LIME's domain-blind Gaussian sampling with
    protocol-aware perturbation that enforces physical network invariants.

    Args:
        schema: The DatasetSchema defining feature metadata and constraints.
        tcp_label_value: For string-encoded protocol columns, the integer
            label representing TCP. Ignored for integer-encoded schemas.
            If None and schema.protocol_encoding is 'string', the user
            must provide this or auto-detection will be attempted.
    """

    def __init__(
        self,
        schema: DatasetSchema,
        tcp_label_value: int | float | None = None,
    ) -> None:
        self.schema = schema
        self.enforcer = ConstraintEnforcer(schema)
        self.tcp_label_value = tcp_label_value

    def generate(
        self,
        x_row: np.ndarray,
        num_samples: int,
        sigma: float | np.ndarray,
    ) -> np.ndarray:
        """Generate a constrained neighborhood around a single instance.

        Args:
            x_row: 1D array of shape (D,) — the instance to explain.
            num_samples: Number of synthetic samples to generate.
            sigma: Perturbation scale. Either a scalar (applied to all features)
                or a 1D array of shape (D,) for per-feature variance.

        Returns:
            2D array of shape (num_samples, D) with the first row equal to x_row.
        """
        d = len(x_row)
        noise = np.random.normal(0, sigma, (num_samples, d))
        neighborhood = x_row + noise

        # Resolve protocol encoding
        encoding = self.schema.protocol_encoding
        protocol_value = None
        tcp_val = TCP_PROTOCOL_INT

        if self.schema.protocol_index is not None:
            protocol_value = x_row[self.schema.protocol_index]
            if encoding == "auto":
                encoding = detect_protocol_encoding(
                    x_row, self.schema.protocol_feature, self.schema.feature_names
                )
            if encoding == "string":
                tcp_val = self.tcp_label_value if self.tcp_label_value is not None else TCP_PROTOCOL_INT

        self.enforcer.enforce(
            neighborhood,
            protocol_value=protocol_value,
            protocol_encoding=encoding,
            tcp_label_value=tcp_val,
        )

        # Pin the first row to the exact original instance
        neighborhood[0, :] = x_row
        return neighborhood
```

# pa_lime/explainer.py
"""Protocol-Aware LIME explainer for NIDS model auditing."""

from __future__ import annotations

import numpy as np
from sklearn.linear_model import Ridge
from sklearn.metrics import pairwise_distances
from sklearn.preprocessing import StandardScaler

from pa_lime.fuzzer import DomainConstraintFuzzer
from pa_lime.result import ExplanationResult
from pa_lime.schemas import DatasetSchema


class ProtocolAwareLIME:
    """Protocol-Aware Local Interpretable Model-Agnostic Explanations.

    Generates local explanations for any black-box NIDS classifier by:
      1. Generating a constrained neighborhood via DomainConstraintFuzzer
      2. Querying the black-box for predictions on the neighborhood
      3. Weighting samples by proximity (exponential kernel)
      4. Fitting a weighted Ridge surrogate
      5. Returning a rich ExplanationResult

    Args:
        schema: DatasetSchema with feature metadata and protocol constraints.
        tcp_label_value: For string-encoded protocol columns, the label-encoded
            integer representing TCP. Ignored for integer-encoded schemas.
        ridge_alpha: Regularization strength for the Ridge surrogate.
    """

    def __init__(
        self,
        schema: DatasetSchema,
        tcp_label_value: int | float | None = None,
        ridge_alpha: float = 1.0,
    ) -> None:
        self.schema = schema
        self.fuzzer = DomainConstraintFuzzer(schema, tcp_label_value=tcp_label_value)
        self.ridge_alpha = ridge_alpha

    def explain_instance(
        self,
        x_row: np.ndarray,
        predict_fn,
        num_samples: int = 5000,
        sigma: float | np.ndarray = 0.1,
        kernel_width: float | None = None,
        class_to_explain: int | None = None,
    ) -> ExplanationResult:
        """Generate a local explanation for a single instance.

        Args:
            x_row: 1D array of shape (D,) — the flow instance to explain.
            predict_fn: Callable that accepts (N, D) array and returns either:
                - 1D array of shape (N,): probabilities or scores for one class
                - 2D array of shape (N, C): probabilities for C classes
            num_samples: Number of neighborhood samples for surrogate training.
            sigma: Perturbation scale. Scalar or per-feature array of shape (D,).
            kernel_width: Width of the exponential proximity kernel.
                If None, uses heuristic: 0.75 * sqrt(D).
            class_to_explain: For multi-class predict_fn returning (N, C),
                which class column to explain. If None and output is 2D,
                explains the class with highest predicted probability at x_row.

        Returns:
            ExplanationResult with coefficients, fidelity, and feature metadata.
        """
        d = len(x_row)

        # 1. Generate constrained neighborhood
        neighborhood = self.fuzzer.generate(x_row, num_samples, sigma)

        # 2. Get black-box predictions
        raw_preds = predict_fn(neighborhood)

        # Handle multi-class output
        predicted_class = None
        if raw_preds.ndim == 2:
            if class_to_explain is None:
                # Explain the argmax class at the query point
                class_to_explain = int(np.argmax(raw_preds[0]))
            predicted_class = class_to_explain
            y_neighborhood = raw_preds[:, class_to_explain]
        else:
            y_neighborhood = raw_preds

        # 3. Compute proximity weights
        scaler = StandardScaler()
        neighborhood_scaled = scaler.fit_transform(neighborhood)
        query_scaled = neighborhood_scaled[0:1]

        distances = pairwise_distances(
            neighborhood_scaled, query_scaled, metric="euclidean"
        ).flatten()

        if kernel_width is None:
            kernel_width = 0.75 * np.sqrt(d)

        weights = np.exp(-(distances ** 2) / (kernel_width ** 2))

        # 4. Fit weighted Ridge surrogate
        surrogate = Ridge(alpha=self.ridge_alpha)
        surrogate.fit(neighborhood_scaled, y_neighborhood, sample_weight=weights)

        r_squared = surrogate.score(
            neighborhood_scaled, y_neighborhood, sample_weight=weights
        )
        local_prediction = float(surrogate.predict(query_scaled)[0])

        # 5. Build result
        return ExplanationResult(
            feature_names=list(self.schema.feature_names),
            coefficients=surrogate.coef_,
            intercept=float(surrogate.intercept_),
            r_squared=float(r_squared),
            predicted_class=predicted_class,
            local_prediction=local_prediction,
            num_samples=num_samples,
        )
```

# pa_lime/metrics.py
"""XAI evaluation metrics for Protocol-Aware LIME explanations."""

from __future__ import annotations

import numpy as np
from scipy.stats import spearmanr

from pa_lime.explainer import ProtocolAwareLIME
from pa_lime.result import ExplanationResult


def semantic_robustness(
    x_row: np.ndarray,
    explainer: ProtocolAwareLIME,
    predict_fn,
    epsilon: float = 0.05,
    n_iter: int = 50,
    num_samples: int = 5000,
    sigma: float = 0.1,
) -> float:
    """Evaluate explanation stability under semantically valid mutations.

    Generates n_iter protocol-valid mutations of x_row (bounded by epsilon),
    computes explanations for each, and measures the mean Spearman rank
    correlation of feature importances against the baseline explanation.

    A score close to 1.0 indicates highly stable (robust) explanations.

    Args:
        x_row: 1D array — the instance to evaluate robustness for.
        explainer: A configured ProtocolAwareLIME instance.
        predict_fn: The black-box model's prediction function.
        epsilon: Maximum perturbation magnitude for generating mutations.
        n_iter: Number of mutations to test.
        num_samples: Samples per explanation (passed to explain_instance).
        sigma: Perturbation scale for each explanation (passed to explain_instance).

    Returns:
        Mean Spearman rank correlation (float in [-1, 1]).
    """
    base = explainer.explain_instance(x_row, predict_fn, num_samples=num_samples, sigma=sigma)
    base_coef = base.coefficients

    scores = []
    for _ in range(n_iter):
        # Generate a single valid mutation using the fuzzer with small epsilon
        mutated = explainer.fuzzer.generate(x_row, num_samples=2, sigma=epsilon)
        x_mut = mutated[1]  # row 0 is the original; row 1 is the mutation

        mut_result = explainer.explain_instance(
            x_mut, predict_fn, num_samples=num_samples, sigma=sigma,
        )
        corr, _ = spearmanr(base_coef, mut_result.coefficients)
        if not np.isnan(corr):
            scores.append(corr)

    if not scores:
        return 0.0
    return float(np.mean(scores))


def fidelity(result: ExplanationResult) -> float:
    """Return the R-squared fidelity of an explanation.

    Args:
        result: An ExplanationResult from ProtocolAwareLIME.

    Returns:
        The R-squared score of the Ridge surrogate.
    """
    return result.r_squared


def sparsity(result: ExplanationResult, threshold: float = 0.01) -> int:
    """Count the number of features with importance above a threshold.

    Lower sparsity (fewer non-zero features) reduces analyst fatigue.

    Args:
        result: An ExplanationResult from ProtocolAwareLIME.
        threshold: Minimum absolute coefficient to count as "active".

    Returns:
        Number of active features.
    """
    return int(np.sum(np.abs(result.coefficients) > threshold))
```

## Summary of decisions

| Decision | Choice | Rationale |
|----------|--------|-----------|
| Package name | `pa_lime` | User specified |
| Location | `pa_lime/` at repo root | Standalone package within repo |
| UNSW-NB15 | Both native (49 Argus features) and CICFlowMeter schemas | User specified |
| Protocol auto-detection | Heuristic: {1,6,17} -> integer, {0,1,2} -> string | IANA vs label-encoding disambiguation |
| Custom schemas | `DatasetSchema` dataclass with all fields | User specified |
| Return type | `ExplanationResult` dataclass with coefficients, fidelity, intercept, feature names, top_features(), as_dict() | User specified rich object |
| Multi-class | `class_to_explain` param; auto-selects argmax if omitted | Supports both 1D and 2D predict_fn |
| Surrogate | Ridge regression (scikit-learn) | Standard LIME choice, handles collinear features |
| Kernel | Exponential with configurable width, default 0.75*sqrt(d) | Standard LIME heuristic |
| Sigma | Accepts scalar or per-feature array | Supports feature-wise variance tuning from paper |
